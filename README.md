# Nand2Tetris Course Project Solutions
My solutions to the projects of the Nand2Tetris course. The playlists for both parts of the course are available on YouTube and the assignments can be attempted from the online IDE offered by the course instructors. Further, the course is based on the book Elements of Computing Systems, by Noam Nisan and Shimon Schocken.

The course is also offered on Coursera, which is not free.
